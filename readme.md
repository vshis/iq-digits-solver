# IQ Digits Solver

This is a solver for the IQ Digits game by Smart Games.

Ref: <https://www.smartgames.eu/uk/one-player-games/iq-digits>

## Requirements

- Python >= 3.10.6

For running tests:

- Pytest >= 7.4.3

Install all Python requirements with:

```bash
pip install -r requirements.txt
```
